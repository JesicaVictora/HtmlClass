Una lista consiste en una serie de elementos colocados uno debajo del otro (o de lado)
En HTML tenemos dos tipos de listas, las ol (ordered list) y las ul (unordered list).
Cada una de estas contienen a sus li (list item).
Las listas son un elemento muy útil en htlm, no solo se utilizan para crear listados, sino que también se pueden usar para crear elementos como un menú de navegación.

Lista de la compra con ul

  • Arroz
  • Leche
  • Harina
