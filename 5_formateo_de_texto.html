El formateo de texto se utiliza para resaltar partes de nuestra página web, incluso puede hacer que sea más accesible.
La etiqueta b pondrá nuestro texto en negrita, pero para que sea más accesible se recomienda el uso de strong
La etiqueta i convertirá nuestro texto en itálica, pero para generar más accesibilidad se recomienda el uso de em
Para subrayar el texto se utiliza la etiqueta U.
