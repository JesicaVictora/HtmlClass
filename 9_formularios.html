Los formularios son muy útiles para interactuar con los usuarios.
Como por ejemplo para iniciar sesión o para que se pongan en contacto contigo.
Los formularios deben estar inicializados por la etiqueta form, y dentro de ella los elementos de entrada que lo componen. Hay 3 tipos de elementos de entrada, input, select y textarea.
A cada elemento se le puede agregar un texto de referencia con la etiqueta label, esta misma se debe enlazar al elemento que corresponde. Al enlazarla no solo se mejora la accesibilidad de la web, sino que también se amplía el área de interactividad del campo, ya que al hacer clic sobre el texto se seleccionara el elemento correspondiente.
Para generar dicho enlace, el valor en el atributo for de la etiqueta label debe coincidir con el valor del atributo id del elemento.
Para indicar el valor del elemento se debe agregar el atributo value.
En el caso de algunos input como el tipo texto, email, etc. se les puede poner un value por defecto, el cual dejara de tener relevancia si el usuario ingresa un valor.
Para especificar en que campo de la base de datos se debe guardar el valor, debemos ponerle un name al elemento.
Y por último, para poder enviar el formulario debemos incluir un submit.
